# Sokoban Solver using A*

A Sokoban puzzle solver implementing classical search algorithms with an interactive graphical interface.

## Features

* A* Search
* Breadth-First Search (BFS)
* Depth-First Search (DFS)
* Custom heuristic
* Interactive GUI
* Manual and AI play modes
* Map loading from files
* Execution time and move count

## Technologies

* Python
* Numpy
* Pygame

## How to Run

```bash
pip install -r requirements.txt
python main.py
```
